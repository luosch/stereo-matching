## Language
c++11
## Dependencies
```
cmake
opencv
```
## Generate the results
```
cmake .
make
./runner
```
## Directory structure
```
.
├── README.md
├── main.cpp (program entry)
├── StereoMatching.hpp (implement SSD, NCC and ASW)
├── ALL-2views (source image)
└── result (output image)
```
